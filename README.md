# practica-1

## Objectiu
Controlar un LED des del Monitor Sèrie mitjançant ordres "ON" i "OFF".

## Autors
- Xavier Aguilar
- Dario Brea

## Materials
- ESP32
- LED
- Resistència 220Ω

## Funcionament
1. Connectar l'ESP32 per USB
2. Obrir el Monitor Sèrie a 115200 baud
3. Escriure "ON" per encendre el LED
4. Escriure "OFF" per apagar el LED

- `/src` - Codi font de la pràctica
- `/doc` - Documentació, esquemes, vídeos
